# WineTracker

Create venv inside main project directory:  

wine_tracker/venv/

cd wine_tracker

mkdir venv

sudo pip install -r ../requirements.txt

(enter your sudo pwd)

*******************

To run server:

cd PROJ_BASE_DIR

python manage.py runserver

********************

Working urls so far:

localhost:8000/admin

localhost:8000:buywine -> then can click on "Save New Wine Purchase link
